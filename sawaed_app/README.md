# تطبيق سواعد عربية

تطبيق موبايل (Flutter) لنظام إدارة الموارد البشرية - سواعد عربية.

## الشاشات الحالية
- تسجيل الدخول (رقم الهوية + كلمة المرور)
- الحضور والانصراف بالبصمة (أول شاشة بعد الدخول)
- الإشعارات اللحظية

## طريقة الحصول على APK جاهز (بدون تثبيت أي شيء عندك)

1. ارفع هذا المشروع على مستودع (Repository) جديد في حسابك على GitHub
2. بمجرد الرفع، هيشتغل "روبوت البناء" (GitHub Actions) تلقائيًا
3. بعد 3-5 دقائق تقريبًا، افتح تبويب **Actions** في المستودع
4. هتلاقي عملية بناء (Build) خلصت بعلامة ✓ خضراء
5. ادخلها، وفي الأسفل هتلاقي **Artifacts** واسمه `sawaed-app-release`
6. نزّله (هيكون ملف zip فيه الـ APK جواه)، فك الضغط، وثبّت الـ APK على أندرويد

## ملاحظة مهمة
التطبيق حاليًا شغال بشاشات جاهزة لكن **بدون ربط حقيقي بالسيرفر** (بيانات تجريبية).
لتفعيل الربط الحقيقي بـ hr.sawaedarab.com، لازم نبني نقاط API في Laravel أولاً
(تسجيل دخول، تسجيل حضور/انصراف، جلب الإشعارات) ثم نربط `lib/services/api_service.dart` بيها.

## التعديل لاحقًا
كل شاشة في ملف منفصل داخل `lib/screens/`. الألوان والتنسيق العام في `lib/theme/app_theme.dart`.
